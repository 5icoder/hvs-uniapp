# 二、项目运行

- 安装[HBuilderX](https://www.dcloud.io/hbuilderx.html)
- 导入hvs和hvs-admin，安装项目依赖
- 创建云开发空间，初始化数据库，上传云函数
- 顶部菜单 -> 运行
